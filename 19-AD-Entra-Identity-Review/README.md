# Day 19 — AD and Entra ID Identity Review

## Purpose

This folder documents the identity review completed across Active Directory and Microsoft Entra ID.

The goal was to compare traditional on-prem-style identity management with cloud identity management.

## Scenario

Nexus Retail Group had both Active Directory identity evidence and Microsoft Entra ID identity evidence in the lab.

This review helped confirm how users, groups, and disabled accounts were structured across both identity environments.

## Work Completed

- Reviewed Active Directory OU structure
- Reviewed Active Directory users
- Reviewed Active Directory groups
- Reviewed Microsoft Entra ID users
- Reviewed Microsoft Entra ID groups
- Reviewed disabled user handling
- Captured identity lifecycle evidence
