# Day 18 — Active Directory Helpdesk Support Tasks

## Purpose

This folder documents practical Active Directory helpdesk support tasks completed in the Nexus Cloud Operations Lab.

The goal was to practise common service desk tasks such as password reset, access verification, least privilege review, and disabled user review.

## Scenario

Nexus Retail Group needed basic Active Directory support tasks completed for users across different departments.

These tasks simulated real helpdesk requests that a Tier 1 support technician or MSP technician would handle.

## Work Completed

- Reset an Active Directory user password
- Required the user to change password at next logon
- Verified Finance group membership
- Confirmed a Sales user did not have Finance group access
- Reviewed a disabled ex-employee account
- Documented support tasks in Jira Service Management
