# Day 2 — Azure Foundation and Cost Control

## Purpose

This folder documents the Azure foundation setup for the Nexus Cloud Operations Lab.

The goal was to create an organised Azure environment with clear resource grouping, basic cost awareness, and a structure that could support the rest of the lab.

## Scenario

Nexus Retail Group needed a clean cloud foundation before building networking, identity, storage, monitoring, and server resources.

## Work Completed

- Created Azure resource groups for the lab environment
- Organised resources by project area
- Applied basic naming structure
- Reviewed cost management
- Created budget/cost control evidence
- Prepared the Azure environment for later project phases

## Resource Group Areas

The lab used resource groups to separate major areas such as:

- Identity
- Networking
- Servers
- Security
- Active Directory

## Skills Demonstrated

- Azure portal navigation
- Resource group creation
- Basic cloud resource organisation
- Cost awareness
- Budget review
- Environment planning
