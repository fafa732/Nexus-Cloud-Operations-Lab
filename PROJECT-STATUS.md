## Important Accuracy Notes

This is a portfolio lab, not a production environment.

Nexus Retail Group is fictional.

Jira Service Management is the official service desk platform used from Day 14 onward.

Azure DevOps Boards was used only during the early foundation phase for support-style tracking.

Microsoft Intune and MDM were reviewed but not fully configured because Tenant Status access was blocked with a 401 / No Permission issue.

Endpoint management continued through Active Directory Group Policy as the hands-on fallback.

## Final Outcome

The project demonstrates practical IT support, cloud operations, identity administration, ticket documentation, backup readiness, monitoring, file permissions, and endpoint policy management in a simulated MSP-style support environment.
