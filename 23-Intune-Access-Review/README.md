# Day 23 — Intune Access Review

## Purpose

This folder documents the Microsoft Intune access review completed in the Nexus Cloud Operations Lab.

The goal was to review whether Intune could be used for endpoint management and to document the access limitation encountered.

## Scenario

Nexus Retail Group needed an endpoint management route for user and device policy control.

Microsoft Intune was reviewed first, but access to the Intune tenant/admin status area was blocked with a 401 / No Permission issue.

Because of this, the project continued endpoint management through Active Directory Group Policy on Day 24.

## Work Completed

- Opened the Microsoft Intune admin center
- Reviewed access to Intune admin areas
- Identified a 401 / No Permission access issue
- Documented that Intune/MDM configuration was blocked
- Decided to continue endpoint management using Active Directory Group Policy
- Prepared the route for Group Policy endpoint management on Day 24

## Important Accuracy Note

Microsoft Intune and MDM were reviewed but not fully configured.

This project does not claim completed Intune deployment, device enrolment, compliance policies, configuration profiles, or MDM rollout.

## Skills Demonstrated

- Microsoft Intune admin center navigation
- Endpoint management access review
- Troubleshooting blocked admin access
- Technical decision documentation
- Fallback planning
- Endpoint management awareness
