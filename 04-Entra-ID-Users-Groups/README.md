# Day 4 — Microsoft Entra ID Users and Groups

## Purpose

This folder documents the Microsoft Entra ID identity setup for the Nexus Cloud Operations Lab.

The goal was to practise cloud identity administration by creating users, groups, department-based access structure, and an offboarding example.

## Scenario

Nexus Retail Group needed cloud user accounts and department security groups so that access could be managed using group-based administration.

## Work Completed

- Created Microsoft Entra ID users
- Created department security groups
- Added users to the correct groups
- Reviewed group membership
- Disabled an ex-employee account
- Captured identity management evidence

## Departments Used

- IT
- Finance
- HR
- Sales
- Warehouse

## Skills Demonstrated

- Microsoft Entra ID navigation
- Cloud user creation
- Security group creation
- Group membership management
- User lifecycle management
- Disabled user/offboarding review
- Identity documentation
