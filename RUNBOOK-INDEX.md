# Runbook Index

This file lists the operational runbooks for the Nexus Cloud Operations Lab.

Runbooks are practical step-by-step support documents used to repeat common IT support, identity, access, backup, monitoring, and endpoint management tasks.

## Runbook Purpose

The purpose of these runbooks is to show how common support tasks were documented after being completed in the lab.

These runbooks support:

- Helpdesk troubleshooting
- Identity administration
- Access control review
- Backup readiness
- Monitoring and alerting
- File share permissions
- Group Policy endpoint management
- Jira ticket documentation
